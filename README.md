# Car Ride

THIS PROJECT IS STILL UNDER DEVELOPMENT. A LOT OF FEATURES HAVE NOT BEEN IMPLEMENTED YET.

## Prerequisites

* NodeJS

## Usage

```{bash}
git clone https://github.com/hnthap/car-ride --depth=1
cd car-ride
npm install
npm run dev
```
